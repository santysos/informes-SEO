5. Robots.txt

Error 406 - Bloqueado por Mod Security Critico

Al acceder a /robots.txt el servidor devuelve Error 406 Not Acceptable causado por Mod Security del hosting.

Esto significa que Googlebot posiblemente no puede leer el robots.txt, lo cual puede causar:

  • Google asuma que todo esta permitido (o que nada lo esta, dependiendo del cache)
  • El sitemap declarado en robots.txt no se descubra
  • Errores recurrentes en Search Console

Accion urgente: Contactar al hosting para crear una excepcion en Mod Security para /robots.txt.

6. Problemas Adicionales Detectados

URLs con fbclid Indexadas

Segun el contexto previo, hay multiples URLs con parametro ?fbclid= indexadas en Google. Esto genera contenido duplicado masivo.

Solucion: Agregar en Yoast SEO > Avanzado > "Limpiar URLs de permalinks" o agregar regla en .htaccess para strip del parametro fbclid.

Blog Practicamente Inexistente

El post-sitemap.xml contiene solo 1 articulo ("Las 7 fases de la venta", agosto 2024). Para un instituto de ventas, la ausencia de contenido de blog es una oportunidad enorme desperdiciada.

Sin Hreflang

No se detectaron tags hreflang. Si el sitio tiene audiencia en varios paises hispanohablantes, deberia considerarse.

Page Transition de Elementor

Se detecto <e-page-transition> con animacion tipo "circle". Las transiciones de pagina pueden afectar negativamente Core Web Vitals (LCP, INP) al retrasar el renderizado del contenido visible.
